# ДЗ 29. Counter

За допомогою React зробити сторінку вигляду:

Початкове значення передається в компонент Counter в props.

Кнопка "-" зменшує число на одиницю, кнопка "+" збільшує на 1, кнопка "Reset" скидає значення на початкове.
